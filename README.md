♻️ Waste Management Platform

A modern web application to simplify and streamline waste collection for residential and commercial users. This platform allows users to schedule pickups, select waste types, pinpoint locations on a map,
and enables service providers to manage requests — all through a responsive and user-friendly interface.    

 🚀 Key Features

- 🧭 **Easy Scheduling** – Select waste types and schedule pickup with a simple form.
- 🗺️ **Interactive Map** – Choose exact pickup locations using Leaflet.js.
- ♻️ **Waste Type Options** – General, recyclable, and hazardous waste categories.
- 🧑‍💼 **Provider Dashboard** – Service providers can view, manage, and finalize requests.
- 💬 **Contact Form** – For user feedback and inquiries.
- 📱 **Responsive Design** – Works on desktops, tablets, and smartphones.
  
 📊 Data Sources

- **User Input** – Form data: type of waste, location, schedule.
- **Map Data** – Real-time location selection with Leaflet.js.
- **LocalStorage** – Temporary storage of requests (simulates backend).
- **Static Content** – Predefined waste types and form options.
  
🎯 Goals

- ✅ Enhance user experience
- ✅ Improve collection accuracy
- ✅ Optimize service provider workflow
- ✅ Encourage sustainable waste practices
- ✅ Ensure cross-device compatibility

📁 Tech Stack

- HTML5, CSS3, JavaScript
- Leaflet.js (for maps)
- LocalStorage (mock backend)

🛠️ Usage

1. Clone or download the project.
2. Open `index.html` in any browser.
3. Test features as a user or service provider.
